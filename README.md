# Fruit Roll 3D

## Hakkında (Description)
Bu proje, Unity Game Developer vaka çalışması kapsamında (Monopoly GO konseptinden esinlenerek) geliştirilmiş 3D tek oyunculu bir kutu oyunudur. Oyun; envanter yönetimi, dinamik harita üretimi ve interaktif zar mekaniklerini harmanlamaktadır.

## Özellikler (Features)
- **Envanter Sistemi:** Elma, armut ve çilek toplanabilir. Envanter verileri oyun kapatılıp açıldığında korunacak şekilde kaydedilir.
- **Dinamik 3D Harita:** Kare şeklinde değil, düz bir çizgi (Line) yapısında tasarlanan harita JSON verisi okunarak 3 boyutlu olarak oluşturulur.
- **Zar Mekanikleri:** Kullanıcı girdisine göre 3 boyutlu ortamda fiziksel olarak atılan zarlar, oyuncunun tahtadaki hareketini belirler.
- **Görsel Efektler:** Hiçbir 3. parti araç veya eklenti (DoTween vb.) kullanılmadan tamamen Unity'nin kendi sistemleriyle oluşturulmuş animasyonlar ve partikül efektleri (toplama, zar atma vb.) içerir.
